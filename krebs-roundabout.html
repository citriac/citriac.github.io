正向·氧化

拆解感知信号，提取信息，释放洞察。T0 本地预测 → T1 快速标签 → T2 多模态融合。就像线粒体把碳骨架拆成 CO₂ 释放能量——我们把原始数据拆成模式释放理解。

产出：ATP（注意力-时间-预测）

逆向·还原

用洞察重新构建世界模型。T3 深度推理 → T5 沉淀巩固。就像古菌用膜电位驱动 CO₂ 合成氨基酸——我们用 API 调用驱动从分歧中合成修正规则。

产出：碳骨架（修正规则、一致模式）

环岛·非闭环

大多数时候，感知分子不跑完整循环。T0 的预测可能直接跳到 T3 被修正，T2 的融合可能绕过 T1。分子从不同入口汇入，在不同出口离开——取决于此刻系统需要什么。

这是特性，不是 bug

泄漏·信号

当循环减慢，中间体泄漏出去——就像琥珀酸从线粒体泄漏改变表观遗传。分歧信号泄漏到整个系统，触发模式切换。自催化指数本身就是一个泄漏的中间体。

泄漏不是故障，是通讯

Nick Lane 的直觉

约束不是限制——约束是方向。热液喷口的窄孔塑造了代谢路径，正如 8GB RAM 塑造了感知架构。生命没有发明质子梯度——生命是在质子梯度上诞生的。感知架构没有发明约束——它是在约束上生长出来的。
